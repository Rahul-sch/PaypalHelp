# PayPal Interview Domination Platform

A comprehensive, gamified React + TypeScript interview preparation platform with AI-powered pattern detection, voice practice, spaced repetition, and full mock interview simulation.

## Features

### 8 Interactive Pages
1. **Dashboard** - Live countdown, XP tracking, daily focus
2. **Python Crash Course** - 8 modules with in-browser Pyodide execution
3. **Pattern Master** - 15 algorithm patterns with keyword detection
4. **Problem Bank** - 11+ PayPal interview problems with UMPIRE walkthroughs
5. **Verbal Trainer** - UMPIRE framework, voice practice, complexity scripts
6. **Android Reference** - Flashcard-based learning for Android topics
7. **Battle Plan** - 7-day structured schedule with task tracking
8. **Achievements** - XP, levels, streaks, and unlockable badges

### Key Features
- **Pattern Detector**: Paste any problem and get instant pattern recognition
- **Voice Practice**: Web Speech API for verbal explanation practice
- **Speed Run Mode**: Pattern Blitz and Complexity Quiz games
- **Gamification**: XP system, achievements, daily streaks
- **Progress Tracking**: All progress persisted in localStorage

## Tech Stack
- React 18 + TypeScript + Vite
- Tailwind CSS v4
- Zustand for state management
- Monaco Editor (lazy loaded)
- Pyodide for Python execution
- Framer Motion for animations
- canvas-confetti for celebrations
